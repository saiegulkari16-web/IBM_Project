---

# 📌 Overview

Money Matters empowers users to make informed financial decisions by combining the power of **IBM Granite**, **RAG**, and **Agentic AI**. The application simplifies complex financial concepts, provides verified information from trusted sources, and delivers real-time financial insights through an intuitive conversational interface.

---

# 🎯 Problem Statement

The rapid growth of digital finance has transformed how people manage money through online banking, UPI, investments, loans, and government financial services. However, the increasing complexity of financial systems, changing regulations, and rising cyber fraud make it difficult for users to access reliable and up-to-date financial guidance.

There is a need for an intelligent platform that provides accurate, trustworthy, and easy-to-understand financial assistance while promoting financial literacy and informed decision-making.

---

# 🎯 Objective

Money Matters is designed to provide personalized, trustworthy, and real-time financial guidance by:

- 💰 Providing personalized financial guidance based on user needs.
- 📚 Retrieving verified financial knowledge using RAG.
- 🌐 Delivering real-time financial news and government updates.
- 🧮 Offering AI-powered financial calculators and planning tools.
- 🤖 Using Agentic AI to intelligently route user queries.
- ☁️ Leveraging IBM Granite and watsonx.ai Runtime for secure AI-powered assistance.

---

# ✨ Features

- 🤖 AI Financial Assistant
- 📚 RAG-based Knowledge Retrieval
- 📰 Daily Brief (Latest Financial News)
- 🌐 Real-time Web Search
- 🧮 Financial Calculators
- 💬 Personalized Conversations
- 📂 Conversation History
- 📎 File Upload Support

---

# 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| IBM Granite Foundation Model | Enterprise-grade Large Language Model for reasoning and financial guidance |
| IBM watsonx.ai Runtime | Secure deployment and execution of Granite models |
| IBM BOB | AI-assisted development for rapid UI and code generation |
| Retrieval-Augmented Generation (RAG) | Retrieves verified information from trusted financial sources |
| Agentic AI | Routes user queries to the appropriate AI workflow |
| Streamlit | Interactive web application frontend |
| IBM Cloud Object Storage | Secure storage for knowledge sources and uploaded files |
| Live Web Search | Retrieves the latest financial news and regulatory updates |
